About This Tool

The numbers created with this tool are estimates only. Feel free to use them as a starting point for your own creations!

The suggested price is derived straight from the table on page 198 of the Genesys core rulebook.

The suggested hard points is derived from the attachments rules found on page 206 of the Genesys core rulebook.

The suggested encumbrance was derived by studying the existing armors as well as the text on page 198 of the Genesys core rulebook.

Specifically, "When you choose a soak value, you should consider a value between 0 and 3…" as well as, "We recommend giving armor an encumbrance value between 2 and 5…"
